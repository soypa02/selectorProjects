Conteudos insanos:

HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar e apresentar conteúdo na web. É a base de praticamente todos os sites que você visita, e oferece novas funcionalidades e melhorias em relação às versões anteriores, como suporte a multimídia, elementos semânticos e APIs mais poderosas.

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação visual de documentos HTML e XML, como páginas web. Basicamente, o CSS controla como os elementos de uma página são exibidos, incluindo cores, fontes, espaçamento, layouts e outros aspectos visuais. Em termos mais simples, o HTML estrutura o conteúdo da página (o que aparece), enquanto o CSS define como esse conteúdo será apresentado (como aparece)

O que é a propriedade Display em CSS

Em CSS, a propriedade display determina como um elemento HTML é renderizado na página. Ela define se o elemento se comporta como um bloco, um elemento em linha ou se é ocultado. A propriedade display é fundamental para o layout e a disposição dos elementos em uma página web.

display caracteristicas
Block Para blocos principais de conteudo, seçoes e elementos q precisam de sua prorpia linha e controle atotal de tamanho. pense em containers e estruturas verticais
Inline Ideal para pequenaos elementos dentro de um fluxo de texto, como links, negritos, italicos , onde voce nao precisasr controlar largura e altura.
flex define um elemento como um container flexível, permitindo o uso do layout flexbox para organizar seus filhos (elementos filhos). O flexbox é uma ferramenta poderosa para criar layouts responsivos e flexíveis, tanto horizontal quanto verticalmente, com controle sobre o alinhamento, espaçamento e redimensionamento dos itens.
